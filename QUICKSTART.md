# 🚀 Quick Start Guide

## ⚡ Start in 30 seconds

### 1. Start Server
```bash
cd /workspaces/Smile-Game-Hud
python3 -m http.server 8080
```

### 2. Open Website
- Visit: **`http://localhost:8080`**
- Website loads with 12 games ready to explore!

## 👤 Try as a Regular User

### Step 1: Sign Up (Optional)
1. Click **"Sign Up"** button (top right)
2. Enter:
   - Name: Your name
   - Email: your@email.com
   - Password: password123
3. Click **"Create Account"** ✅

### Step 2: Browse Games
1. Scroll to **"Explore by Category"**
2. Click any category (Action, RPG, Puzzle, etc.)
3. Games filter automatically

### Step 3: Download a Game
1. Find a game card
2. Click **"⬇️ Download"** button
3. **Free games**: Download instantly
4. **Paid games**: Enter payment info:
   - Any 16-digit number for card
   - Any MM/YY for expiry
   - Any 3-4 digits for CVV

### Step 4: Leave a Review
1. Click **"💬 Reviews"** on any game
2. Click ⭐ stars to rate (1-5)
3. Type your thoughts in text area
4. Click **"Post Comment"** ✅

### Step 5: View Dashboard
1. Click **"Dashboard"** (top right)
2. See: Profile, Downloads, Comments

## 🐛 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Server won't start | Make sure port 8080 is free, use different port |
| Page doesn't load | Check URL is `http://localhost:8080` |
| Admin access fails | Password is `Ciontatenx83` (exact case) |
| Data disappeared | Clearing cache deletes localStorage |
| Images don't show | Use valid direct image URLs |
| Payment stuck | Try different browser or clear cache |
