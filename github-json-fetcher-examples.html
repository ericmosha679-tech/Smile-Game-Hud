🔗 GitHub JSON Fetcher Examples

Fetch and display JSON from GitHub URLs with dynamic updates

📋 List Example

Displays an array of items as an unordered list.

📊 Table Example

Displays an array of objects as a formatted table.

📝 Object Example

Displays a JSON object as a definition list.

⏱️ Auto-Refresh Example

Automatically refreshes data every 10 seconds.

🎨 Custom Renderer

Uses custom rendering function for specific formatting.

⚠️ Error Handling

Demonstrates error handling with invalid URLs.

📖 Implementation Guides

1. Basic List Example

// Create fetcher for list display
const listFetcher = new GitHubJSONFetcher({
  url: 'https://raw.githubusercontent.com/user/repo/main/data.json',
  elementId: 'list-container',
  dataType: 'list',
  showLoadingState: true
});

// Manual refresh
listFetcher.refresh();

2. Table Example

// Create fetcher for table display
const tableFetcher = new GitHubJSONFetcher({
  url: 'https://raw.githubusercontent.com/user/repo/main/users.json',
  elementId: 'table-container',
  dataType: 'table'
});

// Update URL
tableFetcher.updateConfig({
  url: 'https://raw.githubusercontent.com/user/repo/main/new-data.json'
});
tableFetcher.refresh();

3. Auto-Refresh Example

// Auto-refresh every 30 seconds
const autoFetcher = new GitHubJSONFetcher({
  url: 'https://raw.githubusercontent.com/user/repo/main/live.json',
  elementId: 'live-container',
  dataType: 'table',
  refreshInterval: 30000 // milliseconds
});

// Stop auto-refresh
autoFetcher.stopAutoRefresh();

4. Custom Renderer

// Use custom rendering function
const customFetcher = new GitHubJSONFetcher({
  url: 'https://raw.githubusercontent.com/user/repo/main/custom.json',
  elementId: 'custom-container',
  dataType: 'custom',
  customRenderer: (element, data) => {
    // Your custom rendering logic
    const html = data.map(item => `
      <div class="custom-card">
        <h3>${item.name}</h3>
        <p>${item.description}</p>
      </div>
    `).join('');
    element.innerHTML = html;
  }
});

5. Error Handling

// Error handling happens automatically
const fetcher = new GitHubJSONFetcher({
  url: 'https://raw.githubusercontent.com/user/repo/main/data.json',
  elementId: 'container',
  showLoadingState: true // Shows loading indicator
  // Errors are caught and displayed in the container
});

// You can also add try-catch if needed
try {
  await fetcher.fetch();
} catch(error) {
  console.error('Fetch failed:', error);
}

6. GitHub Raw URL Format

// Standard GitHub raw URL pattern:
https://raw.githubusercontent.com/{owner}/{repo}/{branch}/{path/to/file.json}

// Examples:
https://raw.githubusercontent.com/octocat/Hello-World/main/data.json
https://raw.githubusercontent.com/microsoft/vscode/main/package.json
https://raw.githubusercontent.com/torvalds/linux/master/README

// Pro tip: Use 'main' or 'master' as branch name
// For user content repos: https://raw.githubusercontent.com/user/username.github.io/main/data.json
